Pokemon Bot
Telegram-бот для создания и сражения покемонов. Пользователи могут создавать собственных покемонов, получать информацию о них и сражаться с покемонами других пользователей.

Основные команды
/go
Создает нового покемона для пользователя. Бот случайным образом выбирает тип покемона из трех доступных классов:

Pokemon - стандартный покемон

Wizard - покемон-волшебник (имеет шанс применить щит в бою)

Fighter - покемон-боец (имеет супер-атаку)

Каждому пользователю доступен только один покемон.

/info
Выводит подробную информацию о покемоне пользователя, включая:

Имя тренера

Имя покемона

Уровень здоровья (HP)

Силу атаки

Номер покемона

Класс покемона

Также отображает изображение покемона.

/attack
Атака покемона другого пользователя. Для использования команды необходимо:

Ответить на сообщение пользователя, покемона которого вы хотите атаковать

Оба пользователя должны иметь покемонов

Классы покемонов
Базовый класс: Pokemon
Стандартные характеристики атаки и здоровья

Получает данные о покемоне с PokeAPI

Содержит методы атаки и отображения информации

Наследники:
Wizard - имеет 20% шанс применить защитный щит при атаке

Fighter - использует случайную супер-атаку (дополнительный урон 5-15 единиц)

Технические детали
Используемые технологии
Python 3.x

pyTelegramBotAPI - для работы с Telegram Bot API

requests - для взаимодействия с PokeAPI

Настройка бота
Создайте бота через @BotFather в Telegram

Получите токен бота

Создайте файл config.py и добавьте токен:

python
token = 'ВАШ_ТОКЕН_БОТА'
Установите зависимости:

bash
pip install pyTelegramBotAPI requests
Запустите бота:

bash
python main.py
Особенности реализации
Каждому пользователю соответствует один покемон

Данные о покемонах хранятся в словаре Pokemon.pokemons

Изображения и имена покемонов получаются из публичного API

Бой происходит по принципу "здоровье минус сила атаки"

При атаке покемона-волшебника есть шанс, что атака будет заблокирована
